# Карта Февральской революции 1918 года в Петрограде

[![ru](https://img.shields.io/badge/lang-ru-blue.svg)](README.md)
[![en](https://img.shields.io/badge/lang-en-red.svg)](README.en.md)

Интерактивная цифровая карта Петрограда периода Февральской революции 1917 года. События привязаны к конкретным точкам города и точным датам.

## О проекте

Проект предполагает создание интерактивной цифровой карты, где пользователь может:
- Выбирать день на временной шкале (февраль-март 1917 года)
- Видеть на карте события, произошедшие в выбранный день
- Просматривать маршруты перемещений революционных отрядов, войск и демонстрантов
- Получать исторические справки о каждой точке и событии
